Simple Text-to-Speech for WordPress.

== Installation ==

The Simple Text-to-Speech plugin can be installed much like any other WordPress plugin.

1. Upload the plugin ZIP archive file via "Plugins" -> "Add New" in the WordPress admin, or extract the files and upload them via FTP.
2. Activate the Simple Text-to-Speech plugin through the "Plugins" list in the WordPress admin.
